## What is Cloud?

Before cloud computing, we had to rely on expensive physical servers, manual software installations, and hard drives tied to a single desktop, limiting our data access to wherever our devices were physically located.

Now with cloud computing, we can access massive processing power, unlimited storage, and complex applications instantly from any device, anywhere in the world, over a simple internet connection.

### What is AWS?

AWS (Amazon Web Services) is the leading cloud platform, which means it provides computing resources, like storage, databases, and networking, over the internet. Its data centers power about a third of the cloud computing services running every day!
For example, when you stream a Netflix show, check your Airbnb booking, or scroll through Instagram, those apps are using AWS to:
Turn your clicks into actions like playing a video or taking you to a new page (using AWS' compute services)

Send data quickly to users worldwide, even when lots of people are visiting that site (using AWS' networking services)

Store billions of photos, messages, and videos (using AWS' storage services)

Secure your information (using AWS' security services)

Enable smart features like voice assistants, recommendations, and filters (using AWS' machine learning services)

AWS professionals are the experts behind the scenes that are building, maintaining, and securing the cloud applications you use every day.

Some of the fastest growing career paths in cloud include:
   *  Cloud Architects
   *  DevOps Engineers
   *  Security Engineers
   *  Data Engineers
   *  Cloud Engineers
